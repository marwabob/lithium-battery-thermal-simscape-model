Lithium Battery Cell Model with Thermal Effects
Overview

This repository presents an electro-thermal lithium-ion battery cell model developed using Simscape Electrical and Simulink. The model enables detailed cell-level analysis by integrating both electrical dynamics and thermal behavior.

This project is designed for applications in electric vehicles (EV), battery management systems (BMS), and energy storage system research.

Modeling Environment
MATLAB R2024a
Simulink
Simscape Electrical
Model Features
Electrical equivalent circuit modeling
State of Charge (SOC) estimation
Internal resistance representation
Heat generation due to electrical losses
Thermal mass and heat transfer modeling
Electro-thermal coupling
Simulation Capabilities
Dynamic load testing
Temperature rise analysis
SOC tracking
Performance evaluation under varying operating conditions
Applications
Electric Vehicle battery systems
Thermal management studies
Energy storage system design
BMS algorithm validation
Academic and research simulations
Tutorial and Demonstration
